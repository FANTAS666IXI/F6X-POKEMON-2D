# _F6X POKEMON 2D_

![THUMBNAIL](Resources/Img/Thumbnail.png)

- **Description :**
  - **Pokemon 2D game like in Unity.**
  - **This project is for learning purposes only.**

---

- **Controls :**
  - **Move Up : `W / ↑`**
  - **Move Down : `S / ↓`**
  - **Move Right : `D / →`**
  - **Move Left : `A / ←`**
  - **Exit Game : `Escape`**

---

- **Stack :**
  - **F6X POKEMON 2D** : `0.7.0`
  - **Unity** : `2019.3.6f1`
