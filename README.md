# _F6X POKEMON 2D_

![THUMBNAIL](Resources/Img/Thumbnail.png)

- **DESCRIPTION :**
  - **Pokemon 2D game like in Unity.**
  - **This project is for learning purposes only.**

---

- **CONTROLS :**
  - **Move Up : `W / ↑`**
  - **Move Left : `A / ←`**
  - **Move Down : `S / ↓`**
  - **Move Right : `D / →`**
  - **Developer Mode : `KEYPAD 0`**
  - **Exit Game : `ESCAPE`**

---

- **STACK :**
  - **F6X POKEMON 2D** : `0.7.3`
  - **UNITY** : `2019.3.6f1`
  - **CONSOLE LOG SYSTEM** : `1.1.1`
